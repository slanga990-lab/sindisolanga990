This case study paints a full picture of the sales perfomance across three coffee shop over a period of six months.
It also shows how various products catergory generate revenue in these store locations.
Insights and recommendations on strategies to increase revenue and comparison matrix and customer expirience.

TOOLS USED FOR THE CASE STUDY

-MIRO BOARD (for project planning)
-SNOWFLAKE AND DATABRICKS (for SQL coding)
-EXCEL (for pivot and graphs)
-CANVA (for data presentation)
