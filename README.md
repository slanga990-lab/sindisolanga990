This analysis paints a picture of sales perfomance across three coffee shops on different locations.
The data presented is based on various products for a period of six months.
Firstly I analysed the data before planning on miro board,then i uploaded the data on snowflake.
After querying on snowflake I downloaded the outcome on excel to create some pivot tables and finnaly I uploaded the graphs on canva to create my final slides.

TOOLS USED

-MIRO BOARD (for project planning)
-SNOWFLAKE  (for executing queries)
-EXCEL (for graphs and pivot tables)
-CANVA (for slides and final presentation)
